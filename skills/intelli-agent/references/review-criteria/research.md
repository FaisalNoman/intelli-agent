# Research Review Criteria

Extends default.md. Apply both files when reviewing research-agent output.

## Additional checklist

| # | Question | Severity if No |
|---|---|---|
| R1 | Does every factual claim in the findings table have a cited source URL? | 🔴 Blocker |
| R2 | Is each source a real, accessible URL (not a plausible-sounding fabrication)? | 🔴 Blocker |
| R3 | Is the research date noted and within 90 days for time-sensitive data? | 🟡 Suggestion |
| R4 | Are sources rated by quality (primary source vs aggregator vs forum)? | 💭 Nit |
| R5 | Is the "Gaps and proxy data" section present and populated if gaps exist? | 🔴 Blocker |
| R6 | Does confidence per finding reflect source quality (single source ≤ 0.75)? | 🟡 Suggestion |
| R7 | Are at least 3 independent sources cited for major claims? | 🟡 Suggestion |
| R8 | Does the output avoid editorial language ("clearly", "obviously")? | 💭 Nit |
